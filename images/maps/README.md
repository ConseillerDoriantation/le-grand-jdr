# 🗺️ Cartes du VTT

Dépose ici les **fonds de carte** (scènes) du VTT.

## Ajouter une carte

1. Dépose ton image dans ce dossier `images/maps/` (drag-drop sur GitHub → **Add file →
   Upload files**, ou commit/push en local). Nom simple, sans espace ni accent.
2. URL publique :
   ```
   https://conseillerdoriantation.github.io/le-grand-jdr/images/maps/NOM-DU-FICHIER.jpg
   ```
3. Dans le **VTT** (en MJ) : barre d'outils → bouton **🔗 URL** → colle l'URL.
   La carte s'ajoute au fond **et** à ta bibliothèque de cartes (réutilisable d'un clic).

## Astuces

- Sous-dossiers pour t'organiser : `images/maps/donjons/`, `images/maps/villes/`…
- Très grosse carte (> 4 Mo) : compresse-la en `.webp` ou `.jpg` ~85 % avant de la
  déposer → chargement plus rapide en séance, sans perte visible.
