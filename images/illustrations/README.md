# 🎨 Illustrations des personnages

Dépose ici les **illustrations de profil** et les **photos de galerie** des personnages
(images HD que tu veux garder en pleine qualité).

## Ajouter une illustration

1. Dépose ton image dans ce dossier `images/illustrations/` (drag-drop sur GitHub → **Add
   file → Upload files**, ou commit/push en local). Nom simple, sans espace ni accent.
2. URL publique :
   ```
   https://conseillerdoriantation.github.io/le-grand-jdr/images/illustrations/NOM-DU-FICHIER.jpg
   ```
3. Dans l'app, colle l'URL via le bouton **🔗 URL** :
   - **Illustration de profil** : fiche du perso → onglet **Profil** → changer l'illustration.
   - **Galerie photos** : édition du joueur → section **Photos additionnelles** →
     « 🔗 Ajouter une photo par URL ».

## Astuces

- Sous-dossiers possibles : `images/illustrations/pj/`, `images/illustrations/pnj/`…
- Les **petites** images (portraits cadrés, emotes, icônes boutique/bestiaire) restent
  gérées automatiquement par l'app (base64) — pas besoin de les déposer ici. Ce dossier
  ne sert qu'aux **grandes** illustrations.
